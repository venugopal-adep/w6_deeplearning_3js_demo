Gradient Descent Explained

Gradient Descent is an optimization algorithm used to find the minimum of a function. In this demo, we're minimizing the quadratic function f(x) = x².

Key Concepts:

  • Objective Function: f(x) = x². This is the function we want to minimize.
  • Gradient: f'(x) = 2x. This is the derivative of our objective function.
  • Learning Rate (α): Determines the size of steps we take in each iteration.
  • Update Rule: xnew = xold - α · f'(xold). This is how we update our position in each iteration.

How it works:

  1. Start at an initial point (xinit).
  2. Calculate the gradient at that point.
  3. Move in the opposite direction of the gradient (because we're minimizing).
  4. Repeat steps 2-3 for a set number of iterations.

Interpretation:

  • The blue curve represents our objective function f(x) = x².
  • The red dots show the path of gradient descent.
  • Watch how the algorithm converges towards the minimum (x = 0) as iterations increase.

Experiment:

Try different initial points, learning rates, and iterations to see how they affect the optimization process!

Tips:
  • A small learning rate (e.g., 0.01) makes slow but steady progress.
  • A large learning rate (e.g., 0.9) may overshoot or oscillate.
  • Starting far from zero requires more iterations to converge.
